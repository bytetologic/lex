# My Apexo App

A new project created with [Apexo](https://github.com/bytetologic/apexojs) - TypeScript-first schema validation with seamless static type inference.

## Getting Started

This project uses Apexo for runtime type validation. Check out `index.js` for examples.

### Run the Examples

```bash
node index.js
```

## What is Apexo?

Apexo is a TypeScript-first validation library that lets you:
- ✅ Validate data types at runtime
- ✅ Infer static types from your schemas
- ✅ Build type-safe applications with confidence

## Available Functions

From `@apexo/type`:
- `isString(value)` - Check if value is a string
- `isNumber(value)` - Check if value is a number
- `isBoolean(value)` - Check if value is a boolean
- `isArray(value)` - Check if value is an array
- `isObject(value)` - Check if value is an object
- `isFunction(value)` - Check if value is a function
- `isNil(value)` - Check if value is null or undefined
- `isEmpty(value)` - Check if value is empty
